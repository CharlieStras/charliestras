## Common prefixes for objects

| Prefix                    | Description                                                                                          |
| ------------------------- | ---------------------------------------------------------------------------------------------------- |
| w\_                       | Window                                                                                               |
| m\_                       | Menu                                                                                                 |
| d\_                       | DataWindow                                                                                           |
| pipe\_                    | Data Pipeline                                                                                        |
| q\_                       | Query                                                                                                |
| n\_ or n_standardobject\_ | Standard calss user object, where standardobject represents the type of object; for example, n_trans |
| n\_ or n_cst              | Custom class user object                                                                             |
| u\_ or u_standardobject\_ | Standard visual user object, where sandardobject represents the type of object; for example, u_cb    |
| u\_                       | Custom visual user object                                                                            |
| f\_                       | Global function                                                                                      |
| of\_                      | Object-level function                                                                                |
| s\_                       | Global structure                                                                                     |
| str\_                     | Object-level structrue                                                                               |
| ue\_                      | User event                                                                                           |
